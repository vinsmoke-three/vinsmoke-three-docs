# Claude Code 組み込みコマンド一覧

Claude Code CLIで利用可能なスラッシュコマンド（`/` で始まるコマンド）の一覧です。

---

## 会話管理

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/clear` | `/reset`, `/new` | 会話履歴をクリアし、コンテキストを解放する |
| `/compact [instructions]` | — | 会話をコンパクト化する。オプションでフォーカスする内容を指定可能 |
| `/branch [name]` | `/fork` | 現在の会話の分岐（ブランチ）を作成する |
| `/rewind` | `/checkpoint` | 会話やコードを以前の状態に巻き戻す |
| `/resume [session]` | `/continue` | セッションIDまたは名前で過去の会話を再開する |
| `/rename [name]` | — | 現在のセッション名を変更する |
| `/export [filename]` | — | 現在の会話をプレーンテキストとしてエクスポートする |
| `/copy [N]` | — | 直前のアシスタントの応答をクリップボードにコピーする |
| `/context` | — | 現在のコンテキスト使用量をカラーグリッドで可視化する |

## モデル・パフォーマンス設定

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/model [model]` | — | AIモデルを選択・変更する |
| `/fast [on\|off]` | — | 高速モードを切り替える（同じモデルで高速出力） |
| `/effort [low\|medium\|high\|max\|auto]` | — | モデルの推論レベルを設定する |

## プロジェクト・ワークスペース

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/init` | — | プロジェクトにCLAUDE.mdガイドを初期化する |
| `/add-dir <path>` | — | 現在のセッションに新しいワーキングディレクトリを追加する |
| `/diff` | — | コミットされていない変更をインタラクティブなdiffビューアで表示する |
| `/memory` | — | CLAUDE.mdメモリファイルを編集し、自動メモリを管理する |

## 設定・カスタマイズ

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/config` | `/settings` | 設定インターフェースを開く |
| `/permissions` | `/allowed-tools` | パーミッションを表示・更新する |
| `/hooks` | — | ツールイベントのフック設定を表示する |
| `/keybindings` | — | キーバインド設定ファイルを開く・作成する |
| `/theme` | — | カラーテーマを変更する |
| `/color [color\|default]` | — | 現在のセッションのプロンプトバーの色を設定する |
| `/statusline` | — | Claude Codeのステータスラインを設定する |
| `/vim` | — | VimモードとNormalモードを切り替える |
| `/sandbox` | — | サンドボックスモードを切り替える |
| `/terminal-setup` | — | ターミナルのキーバインドを設定する |
| `/chrome` | — | Chrome連携の設定を行う |
| `/extra-usage` | — | レート制限時の追加使用量を設定する |
| `/privacy-settings` | — | プライバシー設定を表示・更新する（Pro/Maxプランのみ） |

## エージェント・タスク

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/agents` | — | エージェント設定を管理する |
| `/plan` | — | プロンプトから直接プランモードに入る |
| `/tasks` | — | バックグラウンドタスクを一覧表示・管理する |
| `/schedule [description]` | — | クラウドスケジュールタスクの作成・更新・一覧・実行 |

## 統合・連携

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/ide` | — | IDE連携を管理しステータスを表示する |
| `/mcp` | — | MCPサーバー接続とOAuth認証を管理する |
| `/desktop` | `/app` | 現在のセッションをClaude Code Desktopアプリで続行する |
| `/mobile` | `/ios`, `/android` | Claude モバイルアプリのダウンロード用QRコードを表示する |
| `/remote-control` | `/rc` | セッションをリモートコントロール可能にする |
| `/remote-env` | — | Webセッション用のデフォルトリモート環境を設定する |
| `/install-github-app` | — | Claude GitHub Actionsアプリをセットアップする |
| `/install-slack-app` | — | Claude Slackアプリをインストールする |

## アカウント・使用状況

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/login` | — | Anthropicアカウントにサインインする |
| `/logout` | — | Anthropicアカウントからサインアウトする |
| `/cost` | — | トークン使用量の統計を表示する |
| `/usage` | — | プランの使用制限とレート制限の状況を表示する |
| `/status` | — | 設定インターフェースを開く（ステータスタブ） |
| `/stats` | — | 日次使用量、セッション履歴、設定の可視化 |
| `/upgrade` | — | アップグレードページを開く（Pro/Maxプランのみ） |

## ヘルプ・情報

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/help` | — | ヘルプと利用可能なコマンドを表示する |
| `/doctor` | — | Claude Codeのインストールを診断・検証する |
| `/feedback [report]` | `/bug` | Claude Codeに関するフィードバックを送信する |
| `/release-notes` | — | 完全な変更履歴を表示する |
| `/passes` | — | Claude Codeの無料1週間パスを友人に共有する |
| `/stickers` | — | Claude Codeステッカーを注文する |
| `/insights` | — | Claude Codeセッションの分析レポートを生成する |
| `/skills` | — | 利用可能なスキルを一覧表示する |
| `/security-review` | — | 保留中の変更のセキュリティ脆弱性を分析する |

## その他

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/btw <question>` | — | 会話に追加せずにサイドクエスチョンをする |
| `/exit` | `/quit` | CLIを終了する |
| `/plugin` | — | Claude Codeプラグインを管理する |
| `/reload-plugins` | — | すべてのアクティブなプラグインを再読み込みする |
| `/pr-comments [PR]` | — | GitHub PRのコメントを取得・表示する |

---

!!! info "補足"
    - 一部のコマンドはPro/Maxプランのみ利用可能です
    - MCPサーバーは `/mcp__<server>__<prompt>` 形式のカスタムコマンドを提供できます
    - スキルを使ってカスタムコマンドを作成することも可能です
    - CLIで `/` を入力すると、利用可能なコマンドの一覧がフィルタ表示されます
